1. Выведите список студентов физико-математического факультета.
Моё решение:
SELECT s.FIO
FROM G g, Kafedra k INNER JOIN Student s
ON s."Group" = g."Group" AND k.Kafedra = g.Kafedra AND k.Decanat = 'Физико-математический'

Решение ChatGPT(далее соблюдаю его концепцию):
SELECT Student.FIO
FROM Student
JOIN G ON Student."Group" = G."Group"
JOIN Kafedra ON G.Kafedra = Kafedra.Kafedra
WHERE Kafedra.Decanat = 'Физико-математический'

3. Выведите кафедры, список студентов, список студентов в алфавитном порядке.
SELECT Student.FIO, G.Kafedra
FROM Student
JOIN G ON Student."Group" = G."Group"
GROUP BY 1

4. Вывести номера зачеток только студентов физико-технического факультета.
SELECT Zachetki.N_Z, Kafedra.Decanat
FROM Zachetki
JOIN Student ON Zachetki.id_Studenta = Student.ID_St 
JOIN G ON G."Group" = Student."Group" 
JOIN Kafedra ON G.Kafedra = Kafedra.Kafedra
WHERE Kafedra.Decanat = 'Физико-технический'
GROUP BY 1

5. Выведите студентов физико-технического факультета сдавших иностранный язык на 5.
SELECT Student.FIO, Kafedra.Decanat, Exzamen.Predmet, Exzamen.Ball
FROM Student
JOIN G ON G."Group" = Student."Group"  
JOIN Kafedra ON G.Kafedra = Kafedra.Kafedra
JOIN Exzamen ON Exzamen.id_St = Student.ID_St
WHERE Exzamen.Predmet = 'Иностр. язык' AND Exzamen.Ball = 5
GROUP BY 1

6. Подсчитать сколько различных предметов сдавалось в сессию.
SELECT COUNT(DISTINCT Predmet) RAZNYX_PREDMETOV
FROM Exzamen

7. Напишите запрос, который выводит средний балл по экзаменам Васильевой.
SELECT Student.FIO, Round(AVG(Exzamen.Ball), 2) SREDNIY_BALL
FROM Exzamen
JOIN Student ON Student.ID_St = Exzamen.id_St
     AND Student.FIO LIKE 'Васильева%'

8. Определите сколько человек учится на каждой специальности.
SELECT Special, COUNT(FIO) STUDENTOV
FROM Student 
GROUP BY 1

9. Напишите запрос, который покажет все группы физико-технического факультета.
SELECT "Group"
FROM G
JOIN Kafedra ON Kafedra.Kafedra = G.Kafedra 
    AND Kafedra.Decanat = 'Физико-технический'

10. Вывести список фамилий студентов получивших 5 баллов по дифференциальным уравнениям.
SELECT Student.FIO, Kafedra.Decanat, Exzamen.Predmet, Exzamen.Ball
FROM Student
JOIN G ON G."Group" = Student."Group"  
JOIN Kafedra ON G.Kafedra = Kafedra.Kafedra
JOIN Exzamen ON Exzamen.id_St = Student.ID_St
WHERE Exzamen.Predmet = 'Дифференциальные уравнения' AND Exzamen.Ball = 5
GROUP BY 1

11. Напишите запрос, который покажет,  сколько экзаменов  сдавал Шутов.
SELECT Student.FIO, COUNT(Predmet) KOLI4_EXZAMENOV_
FROM Exzamen
JOIN Student ON Student.ID_St = Exzamen.Id_St AND Student.FIO = 'Шутов Анатолий Александрович'

12. Выведите название кафедры студентов, не сдавших хотя бы один экзамен.
SELECT G.Kafedra KAFEDRY_S_NESDANNYM_EXAMENOM
FROM G
JOIN Student ON Student."Group" = G."Group"
JOIN Exzamen ON Exzamen.id_St = Student.ID_St
    AND Exzamen.Ball IS NULL

13. Подсчитать количество кафедр на каждом факультете.
SELECT Decanat, COUNT(Kafedra)
FROM Kafedra
GROUP BY 1

14. Подсчитать количество пятерок на физико-математическом факультете.
SELECT Kafedra.Decanat, COUNT(Exzamen.Ball) PYATYOROK
FROM Exzamen
JOIN Student ON Exzamen.Id_St = Student.ID_St
JOIN G ON Student."Group" = G."Group"
JOIN Kafedra ON G.Kafedra = Kafedra.Kafedra
WHERE Exzamen.Ball = 5 AND Kafedra.Decanat = 'Физико-математический'

15. Определите номера зачетных книжек группы ФТ151.
SELECT Student."Group", Zachetki.N_Z
FROM Zachetki
JOIN Student ON Zachetki.Id_Studenta = Student.ID_St
    AND Student."Group" = 'ФТ151'
GROUP BY 1

16. Выведите перечень специальностей физико-математического факультета.
SELECT Student.Special, Kafedra.Decanat
FROM Student
JOIN Kafedra ON Kafedra.Kafedra = G.Kafedra
JOIN G ON G."Group" = Student."Group" 
    AND Kafedra.Decanat = 'Физико-математический'
GROUP BY 1
